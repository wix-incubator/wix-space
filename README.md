# wix-space

## Clone as a @wix/cli template

### With NPM

```bash
npm create @wix/app@latest --template git@github.com:wix-incubator/wix-space.git
```

### With Yarn

```bash
yarn create @wix/app --template git@github.com:wix-incubator/wix-space.git
```
